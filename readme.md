# Домашні фінанси — Керівництво користувача

Система обліку особистих фінансів на базі Odoo 19. Дозволяє вести облік доходів, витрат і переміщень між рахунками, імпортувати виписки з банку та переглядати баланси по місяцях.

## 1. Основні поняття

| Поняття | Опис |
|---|---|
| **Гаманець** | Будь-який фінансовий рахунок: готівка, банківський рахунок, інвестиційний рахунок |
| **Транзакція** | Прихід або витрата в межах одного гаманця |
| **Переказ** | Переміщення коштів між двома гаманцями (підтримує різні валюти) |
| **Категорія** | Тег для транзакції: "Продукти", "Зарплата", "Комунальні" тощо |
| **Проєкт** | Додаткова мітка для доходів (необов'язково) |
| **Період** | Місяць обліку — зберігається як останній день місяця (31.01.2026, 28.02.2026 тощо) |
| **Поточний період** | Активний місяць, встановлений в налаштуваннях. Документи минулих місяців редагувати не можна |

## 2. Перший запуск: налаштування

Рекомендований порядок налаштування з нуля:

### Крок 1 — Встановити поточний період

**Налаштування → Загальні налаштування → вкладка Домашні фінанси**

Встановіть поточний місяць. Це визначає, який місяць є "активним" для введення даних.

### Крок 2 — Створити гаманці

**Домашні фінанси → Налаштування → Гаманці**

Додайте по одному гаманцю на кожен фінансовий рахунок (готівка, картка, депозит тощо).

### Крок 3 — Створити категорії

**Домашні фінанси → Налаштування → Категорії транзакцій**

Додайте категорії витрат (їжа, транспорт, оренда…) та доходів (зарплата, фриланс…).

### Крок 4 — Вносити транзакції або імпортувати виписки

Після базового налаштування можна почати вести облік вручну або налаштувати автоматичний імпорт виписок.

## 3. Гаманці

**Домашні фінанси → Налаштування → Гаманці**

### Типи гаманців

| Тип | Опис |
|---|---|
| Рахунок | Банківський поточний або ощадний рахунок (за замовчуванням) |
| Готівка | Фізичні кошти |
| Інвестиційний | Брокерський або інвестиційний рахунок |

### Поля гаманця

- **Назва** — унікальна назва (наприклад, "Monobank UAH", "Готівка EUR")
- **Тип** — тип рахунку
- **Валюта** — валюта рахунку (UAH, USD, EUR…)
- **Банк** — банк або фінансова установа (необов'язково)

### Статистика на формі гаманця

Відкрийте гаманець — у верхній частині відображаються 4 кнопки зі статистикою за поточний місяць:

- **Доходи** — сума всіх надходжень
- **Витрати** — сума всіх витрат
- **Перекази до** — кошти, що надійшли від інших гаманців
- **Перекази з** — кошти, відправлені на інші гаманці

Натисніть на кнопку — відкриється список відповідних транзакцій/переказів.

### Обмеження

Якщо в гаманці вже є транзакції або перекази — неможливо змінити:
- валюту
- тип гаманця
- архівувати гаманець

## 4. Категорії та проєкти

### Категорії транзакцій

**Домашні фінанси → Налаштування → Категорії транзакцій**

Категорія має тип: **Дохід** або **Витрата**. Назва унікальна в межах одного типу.

Після того як категорія використана в транзакціях — змінити її тип не можна.

### Проєкти

**Домашні фінанси → Налаштування → Проєкти**

Необов'язкова мітка для транзакцій типу "Дохід". Наприклад: "Фриланс-проєкт А", "Продаж авто". Дозволяє групувати доходи за джерелами.

### Банки

**Домашні фінанси → Налаштування → Банки** *(відображаються в налаштуваннях валют)*

Довідник банків/фінансових установ. Прив'язується до гаманця.

## 5. Транзакції

**Домашні фінанси → Документи → Транзакції**

Транзакція — це будь-який рух коштів всередині одного гаманця.

### Типи транзакцій

| Тип | Коли використовувати |
|---|---|
| **Дохід** | Отримали гроші: зарплата, продаж, повернення боргу |
| **Витрата** | Витратили гроші: покупки, послуги, оплата рахунків |

### Поля транзакції

- **Тип** — Дохід або Витрата
- **Гаманець** — з якого рахунку
- **Категорія** — відповідна до типу (показуються тільки категорії доходів або тільки витрат)
- **Сума** — в валюті гаманця
- **Проєкт** — необов'язково, тільки для доходів
- **Період** — місяць транзакції (автоматично округлюється до кінця місяця)
- **Коментар** — довільний текст

### Список транзакцій

- **Зелений колір** — доходи
- **Червоний колір** — витрати
- За замовчуванням відображається поточний місяць + тільки витрати
- Підтримується групування, фільтри, пошук

### Обмеження по періоду

Не можна створити або відредагувати транзакцію в місяці, що передує поточному. Для змін у минулих місяцях потрібно тимчасово відкотити **поточний період** у налаштуваннях.

## 6. Перекази між гаманцями

**Домашні фінанси → Документи → Перекази**

Переказ фіксує переміщення коштів між двома вашими гаманцями.

### Поля переказу

- **З гаманця** + **Сума** — джерело і сума списання
- **На гаманець** + **Сума** — отримувач і сума зарахування
- **Період** — місяць переказу
- **Коментар** — довільний текст

### Мультивалютні перекази

Якщо гаманці у різних валютах (наприклад, UAH → USD), суми "з" і "на" можуть відрізнятись. Введіть фактичну суму зарахування на цільовий рахунок.

Якщо гаманці в одній валюті — суми мають бути однаковими (система перевірить).

### Обмеження

- Не можна перекидати кошти з гаманця на той самий гаманець
- Обидві суми мають бути позитивними

## 7. Імпорт банківських виписок

**Домашні фінанси → Документи → Імпорт банківської виписки**

Дворівневий процес: спочатку розбір файлу, потім конвертація в документи.

---

### 7.1. Налаштування правил імпорту

**Домашні фінанси → Налаштування → Правила виписок**

Перш ніж імпортувати файл — потрібно налаштувати правило для кожного банку/гаманця.

#### Параметри правила

- **Гаманець** — для якого рахунку це правило
- **Формат файлу** — CSV, XLSX або XLS
- **Перший рядок даних** — з якого рядка починаються дані (щоб пропустити заголовки)
- **Колонка призначення** — номер колонки з текстом призначення платежу
- **Колонка коментаря** — номер колонки з додатковим описом (необов'язково)
- **Колонка суми** — номер колонки з сумою (від'ємне = витрата, додатне = дохід)

#### Рядки правил (автокатегоризація)

Додайте рядки для автоматичного розпізнавання типу і категорії:

| Поле | Опис |
|---|---|
| **Ключове слово призначення** | Текст, що шукається в колонці призначення |
| **Ключове слово коментаря** | Текст, що шукається в колонці коментаря |
| **Тип** | Дохід, Витрата або Переказ |
| **Категорія** | Категорія для доходу/витрати |
| **Гаманець-отримувач** | Для переказів — цільовий гаманець |
| **Пріоритет** | Порядок перевірки (менший = вищий пріоритет) |

**Логіка збігу (в порядку пріоритету):**
1. Збіг і призначення, і коментаря
2. Тільки збіг призначення
3. Тільки збіг коментаря

Якщо жоден рядок не спрацював — рядок отримує статус "Помилка" і потребує ручного виправлення.

**Порада:** Кнопка "Зберегти та дублювати" дозволяє швидко скопіювати схожий рядок правила.

---

### 7.2. Процес імпорту

**Домашні фінанси → Документи → Імпорт банківської виписки → Створити**

#### Крок 1 — Завантажити файл

1. Оберіть **Гаманець** і **Правило**
2. Встановіть **Період** (місяць виписки)
3. Прикріпіть файл виписки (`.xlsx`, `.xls`, `.csv`)
4. Натисніть **"Розібрати файл"**

Система зчитає файл і створить рядки імпорту, застосувавши правила автокатегоризації.

#### Крок 2 — Перевірка рядків

Перегляньте список рядків:

| Колір | Статус | Що робити |
|---|---|---|
| Зелений | Сконвертований | Вже перетворений на документ |
| Червоний | Помилка | Виправте вручну: встановіть тип, категорію, суму |
| Без кольору | Чернетка | Готовий до конвертації |

Для рядків з помилкою — відредагуйте поля прямо в таблиці, потім натисніть **"Позначити як виправлений"**.

#### Крок 3 — Конвертація

Натисніть **"Конвертувати рядки в документи"**.

Усі рядки зі статусом "Чернетка" перетворяться на реальні транзакції або перекази.

---

### 7.3. Автоочищення імпортів

Стара імпортна запис автоматично видаляється за розкладом (за замовчуванням — через 2 місяці після їх створення). Налаштовується в **Системних налаштуваннях → Домашні фінанси → Термін зберігання імпортів**.

## 8. Баланс гаманців

**Домашні фінанси → Звіти → Баланс**

Баланс — це **знімок** стану рахунків на кінець кожного місяця. Він не оновлюється автоматично після кожної транзакції, а розраховується вручну або за розкладом.

### Коли перераховувати баланс

- Після масового імпорту виписок
- Якщо цифри в звіті здаються некоректними
- Після ручних правок транзакцій

### Як перерахувати

**Домашні фінанси → Документи → Перерахувати баланс → Відкрити форму → Перерахувати**

Система перебудує всі записи балансу з нуля на основі всіх транзакцій і переказів.

### Перегляд балансу

- **Список** — баланс по гаманцях і місяцях; від'ємні значення виділяються червоним
- **Зведена таблиця** — гаманці по рядках, валюти по стовпцях

## 9. Звіти

### Транзакції — Графік

**Домашні фінанси → Документи → Транзакції → вид "Графік"**

Кругова діаграма розбивки витрат/доходів по категоріях за вибраний місяць. Зручно для розуміння структури витрат.

### Транзакції — Зведена таблиця

**Домашні фінанси → Документи → Транзакції → вид "Зведена таблиця"**

- Рядки: Тип → Валюта → Категорія
- Стовпці: Місяці
- Значення: Сума

Дозволяє порівнювати категорії між місяцями.

### Баланс — Зведена таблиця

**Домашні фінанси → Звіти → Баланс → вид "Зведена таблиця"**

- Рядки: Гаманці
- Стовпці: Валюти
- Значення: Сума та Сума в базовій валюті

Загальна картина всіх рахунків.

## 10. Системні налаштування

**Налаштування → Загальні налаштування → вкладка Домашні фінанси**

| Параметр | Опис | За замовчуванням |
|---|---|---|
| **Поточний період** | Активний місяць обліку. Попередні місяці заблоковані для редагування | Кінець поточного місяця |
| **Термін зберігання імпортів** | Через скільки місяців автоматично видаляти старі записи імпорту (0 = не видаляти) | 2 місяці |

### Важливо про поточний період

- Щомісяця переводьте поточний період на новий місяць вручну
- Якщо потрібно виправити дані в минулому місяці — тимчасово поверніть поточний період назад, внесіть зміни, потім поверніть на актуальний місяць

## Типові сценарії використання

### Щоденний облік вручну

1. **Домашні фінанси → Документи → Транзакції → Створити**
2. Оберіть тип (Дохід/Витрата), гаманець, категорію, суму
3. Збережіть

### Щомісячний імпорт з банку

1. Завантажте виписку з банківського застосунку (формат XLSX або CSV)
2. **Домашні фінанси → Документи → Імпорт → Створити**
3. Оберіть гаманець, правило, завантажте файл
4. "Розібрати файл" → перевірте рядки → "Конвертувати"
5. Перерахуйте баланс: **Документи → Перерахувати баланс**

### Переказ між рахунками

1. **Домашні фінанси → Документи → Перекази → Створити**
2. Вкажіть гаманець-джерело, суму списання
3. Вкажіть гаманець-отримувач, суму зарахування
4. Збережіть
